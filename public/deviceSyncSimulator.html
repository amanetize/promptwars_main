Demo Mode — this is not live hardware

In production, this app is designed for a real mobile/wearable companion app to sync telemetry (app usage, screen time, sleep, activity) to the POST /api/telemetry/sync endpoint using platform APIs such as Android UsageStatsManager, iOS Screen Time, or Apple HealthKit / Health Connect.

No such native app exists in this hackathon build. This page calls the same real endpoint a real device would use, so you can exercise the full coaching/nudge/insight pipeline end-to-end. Every entry submitted here becomes a genuine stored row — the AI features that read it always make a real, fresh model call over whatever is in the database.

See the README's "Device Sync: Design & Demo Note" section for the full data-point list and what's implemented vs. documented-only.
